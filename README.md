# image color palette gen
Simple website to upload images to and get a color palette. Build to understand image processing and file uploads.

## Features
- Generates color palette form images
- Accepts png and jpg

## Tech Stack
Python, Flask, Pillow, numpy

## Installation
```bash
pip install -r requirements.txt
python main.py
```
Server can be reached under [http://127.0.0.1:5000](http://127.0.0.1:5000)
